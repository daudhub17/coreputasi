# 🚀 Coreputasi

> **Coreputasi** (Core + Reputasi) adalah platform portofolio digital terpadu dan pusat edukasi mandiri. Situs ini dirancang untuk menampilkan berbagai bentuk karya—mulai dari *software development*, dokumen/studi kasus, hingga media digital—sekaligus menjadi wadah berbagi catatan belajar dan dokumentasi teknis secara terbuka.

---

## 🌟 Fitur Utama

- **📂 Multi-Format Portfolio Showcase**
  - **Code & Apps:** Proyek web, aplikasi, skrip, dan repositori open-source.
  - **Digital Assets:** Desain UI/UX, grafis, media interaktif, audio/video, dan penulisan konten.
  - **Documents & Case Studies:** Whitepaper, dokumen teknis, laporan proyek, dan modul panduan.
- **📚 Open Knowledge Hub (Pusat Belajar)**
  - **Tech Notes & Cheatsheets:** Catatan ringkas untuk pemrogram dan kreator digital.
  - **Tutorials & Articles:** Panduan *step-by-step* mengenai pengembangan web, desain, dan dokumentasi.
- **👤 Professional Profile & Interactive Resume**
  - Ringkasan keahlian (*Skill Tree*), riwayat pengalaman, dan tautan kontak profesional.

## 🛠️ Tech Stack & Alat Pengembangan

- **Framework:** [Astro](https://astro.build/) / [Next.js](https://nextjs.org/) (Static Site Generation & MDX support)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Content:** MDX (Markdown + React/Astro Components)
- **Deployment:** [Vercel](https://vercel.com/) / [Netlify](https://netlify.com/)
- **Icons & Assets:** Lucide Icons / Heroicons

## 🚀 Memulai (Getting Started)

### Prasyarat
- [Node.js](https://nodejs.org/) (versi 18+ direkomendasikan)
- `npm`, `pnpm`, atau `yarn`

### Instalasi & Jalankan Lokal

1. **Clone Repositori**
   ```bash
   git clone https://github.com/username/coreputasi.git
   cd coreputasi
   ```

2. **Install Dependensi**
   ```bash
   npm install
   # atau
   pnpm install
   ```

3. **Jalankan Server Development**
   ```bash
   npm run dev
   # atau
   pnpm dev
   ```
   Buka browser dan akses `http://localhost:4321` (atau port yang ditentukan).

4. **Build untuk Produksi**
   ```bash
   npm run build
   ```

## 📝 Format Menambahkan Konten Baru

Setiap karya atau modul belajar dapat ditambahkan sebagai file `.md` atau `.mdx` di dalam folder `content/`.

Contoh *Frontmatter* untuk **Portofolio Proyek**:

```markdown
---
title: "Sistem Manajemen Dokumen Digital"
category: "docs" # pilihan: code | digital | docs
date: "2026-07-20"
summary: "Dokumentasi & studi kasus arsitektur sistem informasi berbasis cloud."
tags: ["Documentation", "Architecture", "Cloud"]
coverImage: "/images/projects/doc-system.png"
featured: true
---

# Ringkasan Proyek
Tuliskan detail studi kasus atau deskripsi proyek di sini...
```
